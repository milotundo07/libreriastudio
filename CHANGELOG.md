# Changelog

## 1.0.0

- riscrittura modulare completa;
- migrazione IndexedDB dalla versione precedente;
- validazione checksum ISBN;
- supporto a più esemplari della stessa edizione;
- importazioni transazionali con anteprima e ripristino;
- backup JSON versionato con copertine locali;
- esportazione e importazione Excel;
- cestino;
- OCR assistito con scelta manuale dell’edizione;
- eliminazione di JSONP e proxy CORS pubblici;
- PWA, accessibilità, test e deploy automatico.


## 1.0.2

- distribuzione piatta compatibile con caricamento web su GitHub;
- corretto il blocco totale degli eventi causato dal pulsante di ripristino mancante;
- aggiornato il service worker per evitare asset JavaScript obsoleti.
