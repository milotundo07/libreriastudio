# Sicurezza

## Dati trattati

La biblioteca è conservata localmente in IndexedDB. Le ricerche bibliografiche inviano ISBN o testo di ricerca ai cataloghi esterni indicati nel README.

## Segnalazione di vulnerabilità

Non pubblicare dati personali o backup reali in una issue pubblica. Invia una segnalazione privata al proprietario del repository tramite i canali indicati nel suo profilo GitHub.

## Dipendenze esterne

Le librerie per scanner, OCR ed Excel sono caricate da URL versionati definiti in `src/config.js`. Aggiornarle soltanto dopo aver eseguito i test e verificato manualmente i relativi flussi.
